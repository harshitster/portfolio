About Me

I'm a Graduate ML Systems Researcher at UC San Diego's STABLE Labs under Prof. Jishen Zhao, currently working on distributed systems for vector databases using emerging interconnects in collaboration with SK Hynix. As a second-year MS Computer Science student specializing in AI and Distributed Systems, I'm set to graduate in 2026.

My research journey began at PES University, Bangalore, where I focused on Image Processing and Computer Vision. I've gained valuable industry experience through internships at Peptris Technologies, working on 2D/3D molecular data with Generative AI, and a Deep Learning Research position at the National University of Singapore.

I'm passionate about pushing the boundaries of scalable AI infrastructure and always excited to collaborate on innovative projects at the intersection of AI, Distributed Systems, and Databases.

DistRAG - Distributed Database System with RAG-powered Querying

A containerized distributed database system that simplifies data access through conversational AI interfaces. The platform transforms complex SQL operations into intuitive natural language interactions while maintaining enterprise-grade scalability and reliability.

DistRAG orchestrates multiple specialized microservices including an AI reasoning engine, vector-based caching layer, and event-driven notification system to deliver sub-second query responses across terabyte-scale datasets.

Technologies: PostgreSQL, Citus, ChromaDB, LlamaIndex, Redis Vector Search, Docker Swarm, FastAPI, NGINX

StreamGrid - Distributed Video Storage System

A distributed video storage and streaming system that demonstrates practical implementations of advanced distributed computing principles. The platform handles large-scale video operations through intelligent load distribution and consensus-based metadata management.

StreamGrid employs a three-tier microservices architecture featuring RAFT-based consensus for metadata consistency and SHA-256 consistent hashing for automatic load balancing with zero-downtime file migration.

Technologies: Go, etcd, gRPC, Protocol Buffers, FFmpeg, MPEG-DASH, SHA-256 Consistent Hashing, RAFT Consensus
GitHub

Distributed Transaction Processing System

A comprehensive distributed systems implementation that guarantees atomic transactions across multiple independent key-value stores. The system addresses the fundamental challenge of maintaining data consistency in distributed environments.

Features a Go-based coordinator orchestrating Two-Phase Commit operations across Rust-implemented backend servers, with extensive automated testing across 13+ failure scenarios ensuring robust ACID compliance.

Technologies: Go, Rust, gRPC, Two-Phase Commit Protocol, Consistent Hashing, Persistent Logging, Automated Testing
GitHub

Features modular architecture with Selective Kernel Feature Fusion, Dual Attention Units, and Recursive Residual Blocks processing features at three distinct scales, achieving 20+ dB PSNR improvements.

Leverages pretrained VGG16 networks to extract 4096-dimensional normalized feature vectors, creating searchable vector databases with optimized search pipeline supporting multiple distance metrics.

Temporal Vector Database

In Progress

A temporal vector database that efficiently tracks semantic evolution using delta-based storage. Instead of storing complete embeddings for every version, stores base snapshots plus incremental deltas to achieve significant storage efficiency improvements while maintaining fast query performance.

Addresses the limitation that existing vector databases treat embeddings as static, making them inefficient for evolving temporal data. Features hybrid base+delta querying with intelligent candidate selection and sparse delta representation for accurate reconstruction.

Technologies: Python, FAISS, HDF5, NumPy, SciPy, FastAPI, Wikipedia API, Delta Compression
